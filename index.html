Integracion con iframe (ocultar pantalla final)

Si usas Signaturit dentro de un iframe, podes ocultar la pantalla final de dos formas complementarias:

  1. callback_url: Ponele una URL tuya (ej: una pagina en blanco o about:blank). Asi el iframe redirige a tu pagina en vez de mostrar la pantalla de Signaturit.
  2. postMessage events: Escucha los eventos que emite Signaturit desde el iframe para cerrar el modal al instante.
// Escuchar eventos de Signaturit en el iframe
window.addEventListener('message', function(event) {
  // Verificar origen de Signaturit
  if (event.origin.includes('signaturit.com')) {
    var data = event.data;
    if (typeof data === 'string') {
      try { data = JSON.parse(data); } catch(e) {}
    }
    // Evento de firma completada
    if (data.event === 'completed' || data.type === 'completed') {
      // Cerrar el modal/iframe
      document.getElementById('tuModalId').style.display = 'none';
    }
  }
});

Tip: Combina ambos metodos. El callback_url evita que se muestre la pantalla final, y el postMessage te permite cerrar el modal desde tu aplicacion.
